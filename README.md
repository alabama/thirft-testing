# thrift-testing
Simple tests with thrift and it's js files

## Install
```
npm install
```

## Run dev
This will run the webpack dev server. Make sure to configure your .env file if needed
```
$ npm run start
```
### Run thrift server
```
$ cd thrift/nodejs
$ node NodeServer
```
### Open Browser and check console
`http://localhost:9000`
### Run thrift nodejs client
```
$ cd thrift/nodejs
$ node NodeClient
```
